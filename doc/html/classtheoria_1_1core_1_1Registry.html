§ createComponent()

Component * Registry::createComponent ( const TypeName &  type,
int  allow_ambiguity = true 
)

Create a component of the specified type. The following type resolution rules are used: 1) If there is only one factory for the specified type, use that. 2) If there are multiple factories and none has been used then use the default factory whose subtype==type 3) If there are multiple factories and none has been used and there is no default, use the first one (unless allow_ambiguity is false) 4) If there are multiple factories use the first one that has already been used

Parameters
typethe type name to use for look up
allow_ambiguityif false treat case (3) above as ambiguous and raise an exception

std::runtime_error if type not found
