Detailed Description

class Config

A hierarchical data structure representing runtime configuration. Config is the result of parsing a file in some format supported by a config builder and resolving variable references using some chain of ConfigVariableResolvers.

Thus when a Config object arrives at a component's init it is fully specified (variable free) provided the variables could be resolved.
